Performance Optimierungen für Shopware

Features

  • Shopware Querystring wird von der CSS entfernt
  • HTML wird komprimiert (Ist bereits in Shopware 5.6 enthalten)
  • HTTP2 Push von CSS und Javascript (Ist bereits in Shopware 5.6 enthalten)
  • Plugin Konfiguration wird gecacht (Ist bereits in Shopware 5.6 enthalten)
  • Purify-CSS Integration

Purify-CSS Integration

Für Purify-CSS wird NodeJS gebraucht. Purify-CSS muss auch global installiert sein. Der Prozess kann mit den Befehl ./bin/console frosh:purify:css ausgeführt werden

Das Plugin wird von der Github Organization FriendsOfShopware entwickelt. Maintainer des Plugins ist: Soner Sayakci
